# Owolabi Oluwashijibomi — Portfolio

Personal portfolio website for AI Creative Producer Owolabi Oluwashijibomi.

## 🚀 Deploy on GitHub Pages

1. **Create a new GitHub repository**
   - Go to github.com → New repository
   - Name it `portfolio` (or `yourusername.github.io` for a root URL)
   - Set it to **Public**

2. **Upload the files**
   - Upload everything in this folder to the repo
   - Make sure `index.html` is at the root level

3. **Enable GitHub Pages**
   - Go to your repo → **Settings** → **Pages**
   - Under "Source", select **Deploy from a branch**
   - Choose `main` branch, `/ (root)` folder
   - Click **Save**

4. **Your site will be live at:**
   - `https://yourusername.github.io/portfolio/`
   - Or `https://yourusername.github.io/` if you named it `yourusername.github.io`

## 📁 File Structure

```
portfolio/
├── index.html          ← Main page
├── css/
│   └── style.css       ← All styles
├── js/
│   └── main.js         ← Animations & interactions
└── README.md           ← This file
```

## ✏️ Customization

- **Add projects**: Edit the `.project-card` blocks in `index.html`
- **Update email/links**: Search for your email in `index.html`
- **Change colors**: Edit CSS variables at the top of `css/style.css`

Built with HTML, CSS & Vanilla JS — no frameworks required.
